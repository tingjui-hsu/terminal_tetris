# Tetris in terminal

## installation

```
$ git clone https://github.com/tingjui-hsu/terminal_tetris.git
$ make
```

## usage

```
$ tetris
```

press `a`/`d`/`s` to move and press `w` to rotate or you can modify these macro to change keys


```c
#define KEY_LEFT   'A'
#define KEY_RIGHT  'D'
#define KEY_DOWN   'S'
#define KEY_ROTATE 'W'
```
